# CLI Reference

| Command | Description |
|---|---|
| `forge init` | Scaffold new project |
| `forge run <task>` | Run agent with harness |
| `forge watch` | Live TUI dashboard |
| `forge replay <id>` | Replay session |
| `forge explain <id>` | Audit report |
| `forge bench` | Run benchmarks |
| `forge improve` | Self-improvement cycle |
| `forge serve` | Start API server |
| `forge doctor` | System check |
| `forge test` | Run test suite |
| `forge validate` | Validate config |
| `forge diff <a> <b>` | Compare sessions |
| `forge config` | View/edit config |
| `forge plugin` | Manage plugins |
| `forge export <id>` | Export session |
| `forge completion` | Shell completions |
