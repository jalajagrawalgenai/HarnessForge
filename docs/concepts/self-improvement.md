# Self-Improvement

Forge's meta-harness implements the Self-Harness loop:

1. **Weakness Mining** — analyze audit trails, cluster failure patterns
2. **Harness Proposal** — generate minimal, targeted edits
3. **Proposal Validation** — regression test, accept only if improvement without degradation

Validated by research: 33-60% pass rate improvement (Self-Harness, arXiv:2606.09498).
